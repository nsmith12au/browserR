A Central Limit Theorem Illustrator.


It is estimated that the percentage of 1, 2, 5, 10, 20, 50, and 100 dollar bills in circulation is 31%, 3%, 7%, 5%, 23%, 4%, and 27% respectively. Imagine drawing repeated samples at random from this population of a given size, where you compute the average dollar value of the bills in each sample.

Hit the RUN button to simulate this process for sample sizes of size 1-30. You will see a different histogram for samples of size 1 through 30 (use the arrows underneath the graphs to scroll through them), and observe that while the distributions do not look at all normal when the sample size is small, the distribution of the sample means looks more and more like a normal distribution as the number of bills sampled at a time increases.

Also notice that the histograms get progrssively narrower (lower variance/standard deviation) as the sample size increases.

# no pec # It is estimated that the percentage of 1, 2, 5, 10, 20, 50, and 100 dollar bills in circulation is # 31%, 3%, 7%, 5%, 23%, 4%, and 27% respectively. # Suppose you pick a random sample of bills and see what the mean bill value is in that sample. The code below will create a dynamic illustration of this problem. centlimsim = function(numsamp, sampsize, dataset) { par(mfrow=c(2,3)) for(j in 1:sampsize) { means=NULL for(i in 1:numsamp) { sampledata = sample(dataset, size=j, replace=TRUE) means = c(means, mean(sampledata)) } hist(means, main=paste("Sample means, taking", j, "bills at a time"), xlab=paste("Sample mean of", j, "bills sampled at random"), xlim=c(min(dataset), max(dataset)),col="green") } } # Let's create a set of bills to sample from. bills = sample(c(1,2,5,10,20,50,100), size=1000000, prob=c(.31,.03,.07,.05,.23,.04,.26), replace=TRUE) # Now, the centlimsim function will draw many samples of a given size, compute the mean of each sample and produce a histrogram of the sample means. # The code below will draw 1000 samples of size 30, sampling from the data in the bills dataset. # You could of course change the number of samples, the sample size, and the data set to sample from as you like. centlimsim(1000, 30, bills)
