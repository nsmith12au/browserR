Here's some of the sample R commands in the preamble to project 2. Your random simulations will be different because they are...well...random.

Just hit the RUN button.

This widget works with most browsers, but it does not work well with the Safari browser. If you have a Mac, it should work fine using using Chrome. If all else fails, the nuclear option is to just install the software by visiting https://www.r-project.org/.

Once you hit the RUN button, the right pane will contain a working R console where you can execute any additional R commands you like. Note that any text after the # symbol is treated as a comment (not executed) by R.
